# Python CLI Calculator

## Overview
This project is a command-line calculator application built using Python.
It follows best software engineering practices including modular design,
input validation, unit testing, and continuous integration.

The calculator supports basic arithmetic operations and provides
a REPL (Read-Eval-Print Loop) interface for continuous user interaction.

---

## Features
- REPL-based command-line interface
- Supports addition, subtraction, multiplication, and division
- Graceful handling of invalid inputs and division by zero
- Fully unit-tested using pytest
- 100% test coverage enforced via GitHub Actions CI
